# 🏛️ Temple Escape Mini Game - 神庙逃亡

> 一款快节奏的无尽跑酷网页小游戏。在神秘神庙中奔跑，躲避障碍，收集金币，挑战你的极限！

## 🎮 在线试玩

**[👉 点击这里开始游戏](https://3q110.github.io/Temple-Escape-Mini-Game/)**

## ✨ 游戏特色

- **三跑道跑酷** - 左右切换跑道躲避障碍
- **跳跃 & 翻滚** - 上跳躲避低障碍，下滑穿越高障碍
- **金币收集** - 旋转金币 + 连击倍率系统
- **四大道具**：
  - 🧲 **磁铁** - 自动吸引范围内金币
  - 🛡️ **护盾** - 抵挡一次致命伤害
  - ⚡ **双倍得分** - 得分翻倍
  - 🚀 **氮气冲刺** - 极速 + 无敌
- **神秘宝箱** 🎁 - 随机获得强力道具
- **连击躲避** - 连续躲避障碍提升连击倍数
- **成就里程碑** - 100m / 500m / 1000m / 2000m / 5000m 解锁成就
- **星空背景** - 闪烁星星 + 神庙装饰
- **粒子特效** - 金币收集、道具激活、奔跑尾迹
- **音效系统** - 纯 Web Audio API 生成，无需外部资源
- **最高纪录** - 本地存储，刷新纪录

## 🕹️ 操作方式

| 操作 | 键盘 | 触屏 |
|------|------|------|
| 左移 | `←` / `A` | 左滑 |
| 右移 | `→` / `D` | 右滑 |
| 跳跃 | `↑` / `W` / `空格` | 上滑 |
| 翻滚 | `↓` / `S` | 下滑 |

## 🚀 本地运行

```bash
# 克隆仓库
git clone https://github.com/3q110/Temple-Escape-Mini-Game.git
cd Temple-Escape-Mini-Game

# 直接用浏览器打开 index.html 即可游玩
# 或使用 Python 简单服务器
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## 📦 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- Canvas 2D 渲染
- Web Audio API 音效
- 零依赖，单文件运行
